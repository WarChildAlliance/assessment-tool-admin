Quick start / Basic usage

The set of questions details feature allows you to download, edit and delete assessment sets of questions. Its main function however is to provide all the features necessary to manage sets of questions' questions, including:

  • Creation

  • Editing

  • Download

  • Cloning

  • Deletion

Feature description

Set of questions details

The Set of questions details page displays set of questions information, and a list of questions.

It provides the following features to manage the set of questions:

  • Edit

  • Delete

  • Download

  • Reorder questions

Edit set of questions

To edit the set of questions, click on the ‘Edit' button:

This will open a dialog to edit the set of questions in question:

After making the necessary changes, you can click on the ‘Save’ button to apply them.

Delete set of questions

To delete the set of questions, click on its ‘Delete’ button.

It will prompt you with a dialog asking if you really want to delete the set of questions.

Click on the ‘Confirm’ button to confirm the deletion.

Download set of questions as PDF

To download a set of questions as a PDF document, click on the ‘Download’ button:

Reorder Questions

To reorder questions within the set of questions, first click on the button ‘Reorder Questions’:

Then drag the desired questions using their drag handle and drop them where you want them to sit in the question order:

Afterwards, validate the reordering by clicking on the ‘Save order’ button:

Questions

On the questions section of the Set of questions details page, it is possible to:

  • Create

  • Edit

  • Download

  • Clone

  • Delete

questions.

Create a question

To create a new question, click on one of the following buttons:

  • SEL (Social and Emotional Learning)

  • Select

  • Input

  • Number Line

  • Drag and Drop

  • Domino

  • Calcul

This will open a dialog corresponding to the type of question you want to create.

Once the required fields are filled in, click on ‘Create’.

The new question will appear on the questions list of the set of questions details page.

Edit a question

To edit a question, click on its ‘Edit’ button on the questions list of the set of questions details page:

This will open a dialog similar to the creation dialog for this type of question.

You can then edit the dialog fields and click on ‘Save’ to apply the changes.

Download question as PDF

To download a question as a PDF document, click on its ‘Download’ button:

Clone a question

To clone (duplicate) a question, click on its ‘Clone’ button:

This will open a dialog similar to the question’s creation dialog, with all the fields from the question you want to clone already filled in.

Edit some fields if you wish to make changes to the new question, then click on ‘Clone’

Delete a question

To delete a specific question click on its ‘Delete’ button:

It will prompt you with a dialog asking if you really want to delete the question.

To delete the question permanently, click on the ‘Confirm’ button
