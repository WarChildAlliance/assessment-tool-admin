Overview

The topic details feature enables teachers/supervisors to edit and download assessment topics, manage and preview questions and reorder them with an easy drag and drop functionality.

Front-end

Location

Assessment Builder âžœ Topic Details

The Topic Details page can be accessed from any section of the Assessment Builder (Your assessments, Public assessments and Archived assessments) by clicking on the topic card inside the Assessment Summary card.

  • Features available for supervisor topics accessed by /assessment-builder/your-assessments or /assessment-builder/archived-assessment:

Back-end

Available endpoints

Topics available endpoints

Endpoints available in AssessmentTopicsViewSet and located in assessments/views.py:

Edit topic

Endpoint

POST /assessments/<assessment id>/topics/<topic id>/

Body

Form Data:

icon: file,
name: string,
description: string,
show_feedback: number,
allow_skip: true,
evaluated: true,
praise: number,
max_wrong_answers: number,
order: number,
archived: true

ViewSet method

AssessmentTopicsViewSet.update

Front-end method

AssessmentService.editTopic(assessmentId: string, topicId: string, topic: any)

Delete topic

Endpoint

DELETE /assessments/<assessment id>/topics/<topic id>/

ViewSet method

AssessmentTopicsViewSet.destroy

Front-end method

AssessmentService.deleteTopic(assessmentId: string, topicId: string)

Get a single topic

Endpoint

GET /assessments/<assessment id>/topics/<topic id>/

ViewSet method

AssessmentTopicsViewSet.retrieve

Front-end method

AssessmentService.getTopicDetails(assessmentId: string, topicId: string)

Add a topic

Endpoint

POST /assessments/<assessment id>/topics/

Body

Form Data:

icon: file
name: string,
description: string,
show_feedback: number,
allow_skip: boolean,
evaluated: boolean,
praise: number,
max_wrong_answers: number,
order: number,
archived: boolean

ViewSet method

AssessmentTopicsViewSet.create

Front-end method

AssessmentService.createTopic(id: string, topic: FormData)
