Quick start / Basic usage

The library section provides details about each assessment, and is divided into two pages:

  • Assessments

  • Single questions

It is only possible to view information on this page, no information is edited here.

Feature description

Assessments page

Location

The Assessments page displays a table with all non-archived assessments and it also contains the features:

  • Assessments table

  • Assessment expanded row

  • Assign an assessment

  • Filters

Assessments table

The assessment table shows all of the unarachived assessments, each column being, in order:

  • The assessment icon

  • ‘Grade’: grade of the assessment

  • ‘Assessment Title’: name of the assessment

  • ‘Invites’: number of students invited to the assessment

  • ‘Plays’: how many times the assessment has been played by a student

  • ‘Score': global average score for the assessment amongst all of the students

The table columns can be sorted in ascending and descending order by clicking on any column title.

Assessment expanded row

Within a row, clicking on the ‘▼' icon will show the assessment’s sets of questions data:

Within the exanded details, it is then possible to navigate between the assessment’s sets of questions:

It is also possible to navigate between the questions of a set by using the question preview’s left and right arrows:

Assign an assessment

To assign an assessment, click on its ‘Assign’ button on the far right of its row:

You can then follow the steps below to assign selected sets of questions:

How to assign sets of questions

1: In the opened dialog you will see students and groups (only groups with enrolled students are displayed)

2: Select one or more students or groups by clicking on their checkbox in the ‘Students’ or 'Groups' list

3: Choose one assessment at a time by clicking on the assessment’s radio checkbox in the 'Assessment' list

4: Then define the the start date and end date of the student access to the sets of questions:

5: Finally, click on ‘Assign’ to assign the set(s) of questions for the defined students and dates:

Filters

You can filter the results displayed in the assessments table on the Assessments page using the chip and drop-down menus, and the search field:

The filters ‘Topic’ and ‘Learning objective’ may only appear if topics and learning objectives exist for the selected grade and subject.

The filter ‘Numbers’ may only appear for the first grade, and for the subject ‘Math’.

At any point you can reset applied filters by clicking on the ‘Reset filters’ button:

Single questions

Location

The Single questions page displays a table with all non-archived assessments and it also contains the features:

  • Questions table

  • Question expanded row

  • Filters

Questions table

The questions table shows all the questions that belong to assessments' sets of questions, each column being, in order:

  • ‘Question name’: question statement / title

  • ‘ID’: learning objective of the set of questions the question belongs to.

  • ‘Plays’: how many times the question has been played by a student

  • 'Invites': number of students invited to answer the question’s question set

  • ‘Speed’: the average answer speed in seconds

  • 'Score': the average score for this question’s answers

  • ‘Created’: creation date of the question

The table columns can be sorted in ascending or descending order by clicking on their column title.

Question expanded row

You can view more details for a specific question by clicking on the ‘▼' icon in its row (or anywhere else on the row):

Filters

You can filter the results displayed in the questions table by using the chip, dropdown menus, and the search field:

The filters are the same as the ones for the Assessments table detailed above.
