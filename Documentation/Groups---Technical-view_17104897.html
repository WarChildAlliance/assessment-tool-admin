Overview

The group feature enables teachers/supervisors to group students, manage their assessments more efficiently, as well as filter student results on a group-per-group basis.

It contains a ‚Äòname‚Äô field, as well as a many-to-one relationship to a supervisor (several groups can belong to one supervisor).

Group-related features

Group-related features

Several GET endpoints of the visualization module allow their responses to be filtered by group, using query parameters (those endpoints are used to display student results on the Dashboard page).
The way those query parameters are handled can be found within the get_queryset method of several ViewSets in visualization/views.py.
