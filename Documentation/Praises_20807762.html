Overview

Praises are images and animations semi-randomly displayed to students when they answer a question correctly inside a topic.

Logic

Conditional display

The probability of praises occurring is closely related to the praise value provided when creating the topic on the admin platform:

praiseProbability = Math.ceil(Math.random() * (this.topic.praise))

When praiseProbability equals 1 the praise dialog is displayed.

The assets shown inside of the praise dialog are then determined semi-randomly.

Asset types

Animation

Video file

The video file used for animations must have a transparent background.

The only formats that currently support transparent backgrounds are FLV, F4V, AVI, MOV and WEBM. MP4 won’t work.

Within praise.dictionnary.ts, you might need to specify different fallback formats for the same animation, because of inconsistent browser support, like so:

    {
        text: 'topics.praise.praise1',
        audio: 'praise/Praise1.ogg',
        animation: {
            src: {
                mov: '/assets/praise/animations/gobee-animation.edited.mov',
                webm: '/assets/praise/animations/Gobee-ani 2_1.webm',
                mp4: '/assets/praise/animations/Gobee-ani 2_1.mp4'
            }
        }
    },

The praise dialog will switch to the subsequent formats specified within the src object if the first one(s) is(are) not well supported.

Spritesheet

It is otherwise possible to use a spritesheet to dislay an animation. If you use this method, keep in mind that sprites must be layed out horizontally without additionnal padding within the spritesheet, and all frames must have the same size.
You can then specify the frame count in praise.dictionnary.ts like so:

    {
        text: 'topics.praise.praise14',
        audio: 'praise/Praise14.ogg',
        animation: {
            src: '/assets/praise/animations/flower_animated.png',
            frameCount: 46,
            loop: false
        }
    },

Image

Within praise.dictionnary.ts, specify an image field with the image source:

    {
        text: 'topics.praise.praise14',
        image: './src/assets/praise/icons/bee-honey.png',
        audio: 'praise/Praise14.ogg'
    },

Audio

Within praise.dictionnary.ts, specify an audio field with the audio source:

    {
        text: 'topics.praise.praise14',
        image: './src/assets/praise/icons/bee-honey.png',
        audio: 'praise/Praise14.ogg'
    },
